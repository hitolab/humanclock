# Code for numerical simulation and raw data in Xiao et al.

## Raw data 
**rawdata.xlsx** is the collection of experimental data underlying all of the figures.

## Kim-Forger model

**code1_bifurcationdiagram.m** for drawing a bifurcation diagram (Supplementary Fig. 5B).
 
**code2_timecourse.m** for showing the time course under some parameter conditions (Supplementary Fig. 5A).
   
**code3_categorize.m** for detecting the type of bifurcation (Fig. 4A and Supplementary Fig. 4). This program produces *catelog.txt*, which contains numerically measured data of period and amplitude.

**code4_makelist.m** for producing *sniclist.txt*, containing the candidate of parameter set for SNIC bifuracation.

**code5_displaysnic.m** for showing the time course under different values of the parameters detected by code4.

**code6_resonance_KF.m** for drawing a diagram of resonance (Fig. 4B and Supplementary Fig. 7A).

## Goodwin model

**code7_goodwin.m** for drawing a bifurcation diagram (Supplementary Fig. 6).

**code8_resonance_Goodwin.m** for drawing a diagram of resonance (Supplementary Fig. 7B).
